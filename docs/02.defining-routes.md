# Defining Routes

We can create our routes with these macros:

- `get()` for GET routes,
- `post()` for POST routes,
- `put()` for PUT routes,
- `del()` for DELETE routes.
- `patch()` for PATCH routes,

All route macros take `app` as their first parameter, followed by the path and handler(s).

They must be defined after creating the app in the entry point, which is the `int main(void)` function in the `main.c` file.

```c
#include "ecewo.h"
#include "our_handlers.h" // Handlers are in there
#include <stdio.h>

int main(void) {
  App *app = ecewo();
  if (!app) {
    fprintf(stderr, "Failed to initialize server\n");
    return 1;
  }

  // Define the routes
  del(app, "/delete", sample_delete_handler);
  put(app, "/update", sample_update_handler);
  patch(app, "/edit", sample_edit_handler);
  post(app, "/create", sample_create_handler);
  get(app, "/", sample_get_handler);

  if (server_listen(app, 3000) != 0) {
    fprintf(stderr, "Failed to start server\n");
    return 1;
  }

  server_run(app);
  return 0;
}
```

> [!TIP]
>
> We also can define all of our routes in one function and call it once after creating the app in main function:

```c
#include "ecewo.h"
#include "our_handlers.h" // Handlers are in there
#include <stdio.h>

void register_all_routes(App *app) {
  // Define the routes
  del(app, "/delete", sample_delete_handler);
  put(app, "/update", sample_update_handler);
  patch(app, "/edit", sample_edit_handler);
  post(app, "/create", sample_create_handler);
  get(app, "/", sample_get_handler);
}

int main(void) {
  App *app = ecewo();
  if (!app) {
    fprintf(stderr, "Failed to initialize server\n");
    return 1;
  }

  // Register all the routes
  register_all_routes(app);

  if (server_listen(app, 3000) != 0) {
    fprintf(stderr, "Failed to start server\n");
    return 1;
  }

  server_run(app);
  return 0;
}
```

## Dynamic Parameters

Use `:param` syntax for dynamic segments:

```c
get(app, "/users/:id", user_get_handler);
post(app, "/users/:id", user_post_handler);
```

## Wildcard Routes

```c
#include "ecewo.h"
#include <stdio.h>

void catch_all(Req *req, Res *res) {
  send_text(res, 404, "Not Found");
}

int main(void) {
  App *app = ecewo();
  if (!app) {
    fprintf(stderr, "Failed to initialize server\n");
    return 1;
  }

  // Matches any route not already defined
  get(app, "*", catch_all);

  if (server_listen(app, 3000) != 0) {
    fprintf(stderr, "Failed to start server\n");
    return 1;
  }

  server_run(app);
  return 0;
}
```
